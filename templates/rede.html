🍎 Configuração — iPhone / iPad

1
Conecte ao Wi-Fi da empresa
O iPhone deve estar na mesma rede Wi-Fi do PC servidor.
2
Baixe o certificado de segurança
No Safari, abra: {{ url_http }}/cert
Aparece "Perfil Baixado" → toque Fechar.
{% if tem_ssl %} ⬇️ Baixar Certificado {% endif %}
3
Instale o perfil
Ajustes → Perfil Baixado → toque no perfil → Instalar → confirme.
4
Ative a confiança do certificado
Ajustes → Geral → Sobre → Configurações de Confiança de Certificado
Ative o certificado KitConference.
✓
Pronto — feito uma única vez
Abra o Safari e acesse {{ url_https or url_http }}
QR codes das etiquetas abrem automaticamente no Safari.
