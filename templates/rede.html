🍎 Configuração — iPhone / iPad

1
Conecte ao Wi-Fi da empresa
O iPhone deve estar na mesma rede Wi-Fi do PC servidor.
2
Baixe o certificado de segurança
No Safari, abra ou toque no botão acima: {{ url_http }}/cert
Aparece "Perfil Baixado" → toque Fechar.
3
Instale o perfil
Ajustes → Perfil Baixado → toque no perfil → Instalar → confirme.
4
Ative a confiança do certificado
Ajustes → Geral → Sobre → Configurações de Confiança de Certificado
Ative o certificado KitConference.
✓
Pronto — feito uma única vez
Escaneie o QR iOS acima — abre direto no Safari sem aviso.

🤖 Configuração — Android

1
Conecte ao Wi-Fi da empresa
Mesma rede Wi-Fi do PC servidor.
2
Escaneie o QR Android acima
Abre {{ url_http }} direto no Chrome.
Sem aviso de segurança — conexão HTTP simples.
✓
Pronto para uso
Acesse o admin sempre via {{ url_http }}.
O QR das etiquetas usa HTTPS — se aparecer aviso, toque "Avançado" → "Continuar".

🔥 Firewall do Windows

Se um celular não conseguir acessar, execute como Administrador no CMD:

netsh advfirewall firewall add rule name="Kit Conference" dir=in action=allow protocol=TCP localport=8011,8080
