# Cadastro de Clientes — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Criar tabela `clientes` como fonte de verdade, gerenciar clientes inline na página "Veículos e Clientes", e usar `<select>` de clientes nos formulários de veículo e de kit template.

**Architecture:** Nova tabela `clientes` + módulo `app/clientes.py`. Rota `POST /admin/clientes` para criação inline. Rotas existentes de veículos e templates recebem `clientes` no contexto. Campos de cliente nas templates viram `<select>`. Navbar e título renomeados.

**Tech Stack:** FastAPI, SQLite, Jinja2, vanilla HTML/CSS

## Global Constraints

- Todos os timestamps via `now_brt()` de `database.py` — nunca `datetime.now()` ou `CURRENT_TIMESTAMP`
- Acesso ao banco via `with db() as conn:` apenas
- `render(request, template, ctx)` para templates que estendem `base.html`
- Todas as rotas admin protegidas por `@require_login`
- Lógica de negócio em módulos (`app/`), não em `main.py`
- `import app.clientes as clientes_mod` no topo de `main.py`, junto aos outros imports de módulos

- [ ] **Step 2: Criar `app/clientes.py`**

```python
from database import db, now_brt


def listar() -> list[dict]:
    with db() as conn:
        rows = conn.execute(
            "SELECT id, nome, criado_em FROM clientes ORDER BY nome"
        ).fetchall()
    return [dict(r) for r in rows]


def criar(nome: str) -> int | None:
    nome = nome.strip()
    if not nome:
        return None
    try:
        with db() as conn:
            cur = conn.execute(
                "INSERT INTO clientes (nome, criado_em) VALUES (?, ?)",
                (nome, now_brt())
            )
            return cur.lastrowid
    except Exception:
        return None
```

@app.post("/admin/clientes")
@require_login
async def admin_clientes_post(request: Request):
    form = await request.form()
    nome = str(form.get("nome", "")).strip()
    if not nome:
        return RedirectResponse("/admin/veiculos?erro_cliente=vazio", status_code=302)
    resultado = clientes_mod.criar(nome)
    if resultado is None:

    return RedirectResponse("/admin/veiculos?ok=cliente", status_code=302)
